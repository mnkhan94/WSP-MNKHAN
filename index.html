Newsletter Sign Up

What I want to practice/learn?

I want to practice using photo filters that can be applied on top of the image using CSS3. To accomplish the friendly input box, I want to increase my understanding of the border-radius CSS property.

How this relates to what I have/will learn in WDD.

Newsletter sign ups are one of the crucial features of any business, and thus as web designers and developers it is our responsibility to create a user friendly and clear system that can direct the user to establishing a relationship with the client's product and services.

How might I build this?

  • Add the background image to an <aside> or <section> tag using CSS.
  • Add a black low-opacity filter over the image using CSS3.
  • Add an <h2> tag to indicate the newsletter
  • Add a <p> tag to give more information
  • Add an <input> tag along with the <button>
  • Style the <input> with border-radius. And give the button the coloring it needs.
